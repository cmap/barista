PertCountView

A Backbone.View that shows that number of perturbagens matching a given query. Optionally, sub-category counts are give for the type of perturbagen queried for. This view is frequently paired with a PertCountModel or CellCountModel

basic use:

pert_count_view = new PertCountView();

optional arguments:

  1. {string} bg_color the hex color code to use as the backgound of the view, defaults to #ffffff
  2. {string} well_color the hex color code to use as the backgound of the wells, defaults to #bdbdbd
  3. {string} fg_color the hex color code to use as the foreground color of the view, defaults to #1b9e77
  4. {string} span_class a bootstrap span class to size the width of the view, defaults to "span4"
  5. {string} static_text the static text header to use in the view, defaults to "Reagents"
  6. {array} categories an array of objects to use as categories to display, defaults to []
pert_count_view = new PertCountView({bg_color:"#ffffff", 
                            well_color: "#bdbdbd",
                            fg_color: "#1b9e77",
                            span_class: "span4",
                            static_text: "Reagents",
                            categories: []});
